SQL Nedir?

SQL (Structured Query Language), ilişkisel veritabanlarıyla çalışmak için kullanılan bir sorgu dilidir. Çok yaygın kullanılır ve veri ekleme, silme, güncelleme ve sorgulama işlemleri için idealdir.

SQL ile neler yapabilirim? Yeni tablo oluşturabilir, veri ekleyebilir, sıralama ve filtreleme yapabilir, çok büyük verilerle rahat çalışabilirsin.

CREATE TABLE ogrenciler (
  id INT PRIMARY KEY,
  isim VARCHAR(100),
  yas INT,
  sinif VARCHAR(10)
);

Veri eklemek için:

INSERT INTO ogrenciler (id, isim, yas, sinif)
VALUES (1, 'Ali Yılmaz', 13, '7A');

SQL bilmek, veriyle çalışmak isteyen herkes için temel beceridir!
